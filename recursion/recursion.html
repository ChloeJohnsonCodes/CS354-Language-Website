When a function is called within the same function, it is known as recursion in C++. The function which calls the same function is known as recursive function.

Fibonacci Sequence

The fibonacci sequence is a peculiar series of numbers named after an Italian mathematician called Leonardo Bonacci, Leonardo of Pisa, or Leonardo Bigollo Pisano. Fibonacci series is a series of numbers formed by the addition of the two preceding numbers in the series starting from 0 and 1. Fib(0) = 0 base case 1 and Fib(1) = 1 base case 2. The current number is computed by adding the two previous terms. Example: 0,1,1,2,4,6,10,16 etc. In this case, the first two terms are 0 and 1 which is then added to get the third. The fourth number is computed by adding the second and third. The sequence continues until the nth term is reached. The formula is F(n) = F(n-1) + F(n-2). Recursion is defined in terms of itself or of its type. 

        
#include <iostream>
using namespace std;

int fibonacci(int userInput) 
{

    /**
     * Checks if the base case for the first two terms is zero and one and returns the same value 
     * because there won't be any changes in position zero and one.
     * fib(0) = 0 -base case 1
     * fib(1) = 1 - base case 2
     */
    if (userInput == 0 || userInput == 1) {
        return userInput;
    }
    
    //Adds the two previous values to get the next.
    else {
        return fibonacci(userInput-1) + fibonacci(userInput-2);
    }

}

int main() 
{
    int userInput;
    int x = 0;

    //Displays the prompt to the screen
    cout << "Enter the number of terms you want displayed: ";
    
    //Accepts the user input from the keyboard.
    cin >> userInput;

    for (int i = 1; i <= userInput; i++) {
        
        //Stores the output of the evaluated fibonacci method to the result variable.
        int result = fibonacci(x);
        
        //Displays the result of the user input to the screen as an integer variable.
        cout << result << " ";
        x++;
    }
}
