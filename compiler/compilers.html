Choosing a Compiler

  • The most mature and compatible C++ compiler is on GCC. It's part of the GNU Compiler Collection. Many IDEs (Integrated Development Environments) also include a compiler, but some won't. Users should be aware of this before relying on an IDE to compile their programs.

  • The directions to install GCC compilers can be found at https://en.wikibooks.org/wiki/C%2B%2B_Programming/Compiler/Where_to_get

  • Another compiler that can be used for Mac OS X is Clang. Clang comes with Xcode, so Mac OS X users with Xcode don't need to install anything. Directions to install Xcode on Mac OS X can be found at https://www.freecodecamp.org/news/how-to-download-and-install-xcode/

  • If a quick solution is needed, users can also find an online compiler at https://www.onlinegdb.com/online_c++_compiler
