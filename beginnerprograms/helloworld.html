Hello World!

Usually, the first program written in a new language is the Hello World! program. Implementation of Hello World! can be seen below.

        
            /* Hello World!
             * This program demonstrates a very simple write to
             * standard output.
             */

            #include <iostream>           //Include the iostream library
                
            int main()                          //Declare your main function
            {
                std::cout<<"Hello World!";      //Write "Hello World!" to the standard output

                return 0;                       //End with a return code of 0
            }

        
        
            Output: Hello World!
