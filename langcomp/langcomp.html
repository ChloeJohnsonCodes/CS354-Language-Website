C++ was designed to be a mix of C's low-level funtionality and object orientation. The implementation of these ideas created a very flexible language that can be used for systems programming, database software, and even game development.

The object orientation in C++ is easily comparable to the object orientation found in Java. While C++ may not be as universally portable as Java, both languages allow the use of inheritence, polymorphism, method overloading, and encapsulation when writing classes. Even with these similarities, C++ does have its moments to shine - the language allows multiple inheritence and defining methods outside of a class, for example, unlike Java. C++ also allows a declared object to be a variable, where Java does not.

Where C++ differs even more from Java is its foundational codebase in C. C++ utilizes manual memory management, making it a suitable language for systems programming. C++ also copies C's approach to pointers, pointer arithmetic, structs, unions, and header files.

All this to say, C++ is often seen as "C with classes".

The same program, Student, has been written in C, C++, and Java to demonstrate some of these differences. C makes use of a struct in order to construct a student, while C++ and Java create a class in their own ways.

Student.java

The student program as implemented in Java

        
        public class Student {
          private String major;

          public Student() {
            this.major = "Undecided";
          }

          public String getMajor() {
            return major;
          }

          public void setMajor(String major) {
            this.major = major;
          }

          public static void main(String[] args) {
            Student student = new Student();
            System.out.println(student.getMajor());
            student.setMajor("Computer Science");
            System.out.println(student.getMajor());
          }
        }
        
        
            Output: Undecided 
            Computer Science

Student.c

The student program as implemented in C, ustilizing a struct in place of a class.

        
        #include "stdio.h"
        #include "string.h"
        #include "stdlib.h"

        struct student_object_struct {
          char major[40];
        };
        typedef struct student_object_struct Student;

        Student* CreateStudent() {
          Student * newStudent = (Student *) malloc(sizeof(Student));

          if(newStudent == NULL){
            return NULL;
          }
          strncpy(newStudent->major, "Undecided", 40);

          return newStudent;
        };

        Student* setMajor(Student* student, const char major[]){
          if(student == NULL) {
            return NULL;
          }
          strncpy(student->major, major,40);
          return student;
        };

        char* getMajor(Student* student){
          student->major;
        };

        int main() {
          Student* student = CreateStudent("Undecided");

          if(student == NULL){
            return 1;
          }

          printf("%s \n", getMajor(student));
          setMajor(student, "Computer Science");

          printf("%s", getMajor(student));

          free(student);
 
          return 0;
        }
        
        
            Output: Undecided 
            Computer Science

The student program as implemented in C++.

        
        #include "iostream"

        using namespace std;

        class Student {
          string major = "Undecided";
          public:
            void setMajor (string major);

          major = maj;

        int main() {
          Student* student = new Student();
          cout << student->getMajor() << endl;
          student->setMajor("Computer Science");
          cout << student->getMajor();
