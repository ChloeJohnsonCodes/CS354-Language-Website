C++ Compilers

Author: Chloe Johnson

The most mature and compatible C++ compiler is on GCC. It's referred to as the GNU Compiler Collection. Many IDEs (Integrated Development Environments) will include a compiler, but some won't. Users should be aware of this before relying on an IDE to compile their programs.

Windows

There are two ways that users can install the GCC compiler on their Windows system.

Cygwin:

  1. Follow the directions to install the software at http://www.cygwin.com
  2. During the installation, users will be prompted to select packages. They should select the "gcc-c++" package.
  3. Once Cygwin is done installing, users can click on the icon to reveal the Cygwin command prompt.
  4. Cygwin creates an environment that is similar to a Linux/Unix machine, so users can compile sample programs according to the run and compile instructions.

MinGW + DevCpp:

  1. Go to http://www.bloodshed.net/devcpp.html and download a version of Dev-C++ that includes Mingw/GCC.
  2. Start a C++ project with the compiler.

Mac OS X

There is an IDE specifically for Mac operating systems called Xcode. Xcode uses a compiler known as clang instead of the GCC compiler. Clang can be used to compile C++ programs, or the GCC compiler can be invoked from the Mac Terminal.

Linux

The GCC C++ installation instructions vary for each Linux distribution.

  • For Gentoo, GCC C++ is part of the system core (since everything in Gentoo is compiled)
  • For Redhat, get a gcc-c++ RPM, e.g. using Rpmfind and then install (as root) using rpm -ivh gcc-c++-version-release.arch.rpm
  • For Fedora, install the GCC C++ compiler (as root) by using dnf install gcc-c++
  • For Mandrake, install the GCC C++ compiler (as root) by using urpmi gcc-c++
  • For Debian, install the GCC C++ compiler (as root) by using apt-get install g++
  • For Ubuntu, install the GCC C++ compiler by using sudo apt-get install g++
  • For openSUSE, install the GCC C++ compiler (as root) by using zypper in gcc-c++
